## Protocol Support

- **Modbus RTU** -- RS485 serial (all models)
- **Modbus TCP** -- WiFi / Ethernet
- **BACnet MS/TP** -- RS485 serial (all models)
- **BACnet IP** -- WiFi / Ethernet
- **LON** -- L1 model only (Coming Soon)
- **M-Bus** -- Future consideration

## Specifications

- Latency: < 2 ms bridge time
- Node capacity: 247 devices per bus
- Power: 9-24V DC
- OTA firmware updates
- SlateBAS native integration
