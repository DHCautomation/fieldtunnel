# FieldTunnel Firmware Versions

## v0.2.0 (current)
- ESP-IDF v5.5 FreeRTOS firmware
- Permanent APSTA WiFi mode
- Unique AP SSID per device MAC (FieldTunnel-XXXX)
- mDNS hostname (fieldtunnel-xxxx.local)
- NAT bridge — AP clients get full network access
- Dynamic DNS forwarding — works on any network
- OTA firmware updates via web UI
- Modbus TCP Gateway mode (FC01-FC16)
- Raw TCP Tunnel mode (transparent byte pipe)
- Protocol mode selector UI
- BACnet MS/TP — coming soon
- SunSpec Solar — coming soon

## v0.1.0 (archived)
- Arduino proof of concept
- Basic Modbus TCP→RTU bridge
- Simple web UI
- Proved hardware + protocol stack
